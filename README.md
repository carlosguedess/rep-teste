# Meu Repositorio git para Estudos 

Criando um repositorio localmente (Direto na Máquina) e subindo para o git hub (Remoto)

1- git init
2- git add .
3- git commit -m ""
    Trocar Brach para main - git branch -M main
4- git remote add origin https://github.com/carlosguedess/rep-teste.git
5- Git push -u origin main
